# Sir Nelson IELTS and General English School

Static website for Sir Nelson IELTS and General English School.

## Main file

`index.html` is the homepage.

## Hosting

This site can be uploaded directly to GitHub Pages, Netlify, Vercel, or cPanel.

- GitHub Pages: upload the files and enable Pages from the repository root.
- Netlify: deploy the folder directly; `netlify.toml` is included.
- Vercel: import the repository; `vercel.json` is included.
- cPanel: upload the files into `public_html`.

## Structure

- `index.html` - homepage
- `assets/css/styles.css` - website styles
- `assets/js/forms.js` - WhatsApp form helper
- `assets/` - optimized images and icons
- `sitemap.xml` and `robots.txt` - search engine files
